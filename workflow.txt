User sends code
        ↓
FastAPI backend receives request
        ↓
Code → Embedding Model
(converts code to vector)
        ↓
FAISS Vector Database
(searches similar programming knowledge)
        ↓
Relevant Knowledge Retrieved
(from knowledge base)
        ↓
Code + Retrieved Knowledge
sent to Mistral LLM
        ↓
Mistral analyzes the code
        ↓
LLM generates explanation + hint
        ↓
Response returned to user
